Examples are
* Nimiq https://nimiq.com/
* IPFS-js https://github.com/ipfs/js-ipfs
* Ethereum js https://github.com/ethereumjs
* Bitcoin js https://bitcoinjs.org/
Or in general Blockchains that have a WASM based VM
